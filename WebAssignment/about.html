The plugin that is used in this website is a Cursor Animation JQuery plugin,
It's a really easy plugin to use and configure. To start the installation of
the plugin we begin by adding JQuery to our website, we do this by
typing the source of JQuery website.

Now that we have Jquery we download the Plugin and put it into our js folder,
after that we have to call the Plugin by typing at the bottom
of our document the following:

Now that we have the Plugin called, we can now add some javascript and use
JQuery to target the container we want our changed cursor to be displayed at.
The #ghost is the container we are targetting and thats the container where
all the content is placed within the website, making the cursor change
only in there.

Lastly, What I did is set the opacity to 0.6 in the JQuery plugin, I did this because
the intensity of the color on the cloud effect does not go in pair
with the website background. I changed lowered the opacity
to make it suitable with the website.

Before finishing the configuration, I notived the cloud effect blocks the cursor it'self,
which makes it very frustrating if the user needs to look at the cursor.
So I made the cloud effect move a slight bit to the bottom right,
I did this by adding 15px to the positionY and 15px to the positionX.
After this the plugin was done and ready.
